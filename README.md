# Daily Habit Tracker

A modern habit tracking application built with React, TypeScript, and Tailwind CSS to help users build and maintain positive daily habits.

## Features

- 📝 Create and manage daily habits
- 🎯 Track habit completion and streaks
- 📊 Visual streak graphs
- 🏆 Achievement system
- 💪 Motivational quotes
- 🌅 Time-of-day based habits
- 📱 Responsive design

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- date-fns
- lucide-react

## Getting Started

Install dependencies:

```bash
npm Install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
